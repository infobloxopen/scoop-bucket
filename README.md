# Scoop Bucket for Infoblox Open Source

This repository contains [Scoop](https://scoop.sh) manifests for Infoblox open-source tools.

Manifests are **auto-generated** by [GoReleaser](https://goreleaser.com) on each release — do not edit them manually.

## Setup

```powershell
scoop bucket add infobloxopen https://github.com/infobloxopen/scoop-bucket
```

## Install APX

```powershell
scoop install infobloxopen/apx
```

## Upgrade

```powershell
scoop update apx
```

## Uninstall

```powershell
scoop uninstall apx
```

## Available Manifests

| Manifest | Description |
|----------|-------------|
| `apx`   | API Publishing eXperience CLI |
